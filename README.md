# Bank Management System (Java Console Application)

## 📌 Project Overview

The **Bank Management System** is a Java console-based application developed using Object-Oriented Programming (OOP) principles. It allows users to manage bank accounts through a menu-driven interface.

This project was built to strengthen Core Java concepts such as classes, objects, encapsulation, collections, methods, loops, and business logic implementation.

---

## ✨ Features

* 🏦 Create a Bank Account
* 📋 View All Accounts
* 🔍 Search Account by Account Number
* 💰 Deposit Money
* 💸 Withdraw Money
* 💳 Check Account Balance
* 🗑️ Delete Account
* 📖 Menu-driven Console Interface

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/pallavigowda620/java-bank-management-system.git
```

### 2. Navigate to the source folder

```bash
cd java-bank-management-system/src
```

### 3. Compile the project

```bash
javac Main.java BankAccount.java
```

### 4. Run the application

```bash
java Main
```
